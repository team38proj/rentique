RENTIQUE

Sustainable clothing rental platform. You rent garments for set periods instead of buying them.

PURPOSE
Reduce single use clothing. Extend garment life. Give you access to more styles with less ownership.

KEY FEATURES

* Account signup and login
* Browse and search catalog
* Select rental period
* Secure checkout
* Order tracking
* Return scheduling
* Reviews and ratings
* Notifications for due dates and returns

HOW YOU USE RENTIQUE

1. Create an account.
2. Browse the catalog and open a product page.
3. Choose size and rental dates.
4. Add to basket and check out.
5. Receive the item.
6. Wear during your booked period.
7. Pack and return by the due date.
8. Receive confirmation after inspection.

SUSTAINABILITY

* Encourages shared use of garments.
* Lowers demand for new production.
* Keeps clothing in circulation for longer.
* Encourages people to look after their clothing
